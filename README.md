# PoulpyOS
A toy OS for learning

## Compiling for dev
```console
user@debian:~$ make debug
```

## Testing only
```console
user@debian:~$ sudo apt update
user@debian:~$ sudo apt install qemu qemu-system-x86
user@debian:~$ make run
```
